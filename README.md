# Barsotion BarsikOS v2.12 (04.12.2023)

A simple multitask operation system for 'Laulaja-4' computer based on 8085

What is new?
- Добавлена драйвер твердотельного накопителя (SPIFlash W25Q64). Испытаны функции чтения ID, записи и стирания 4кБ-секторов
